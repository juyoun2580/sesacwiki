1. SELECT 기본 문법

SELECT는 데이터베이스에서 데이터를 조회할 때 사용하는 가장 기본적인 SQL 명령어입니다. 테이블에서 원하는 컬럼을 선택해 결과를 가져옵니다. SELECT 뒤에 원하는 컬럼명을 쉼표로 구분하여 나열하고, FROM 뒤에 조회할 테이블명을 지정합니다.

SELECT 컬럼명1, 컬럼명2
FROM 테이블명
WHERE 조건; -- 조건에 맞는 행만 반환

2. WHERE 조건절

WHERE 절은 조회할 데이터의 조건을 지정합니다. WHERE 절은 조건에 맞는 행(row)만 결과로 반환합니다. 다양한 비교 연산자(=, !=, >, <, BETWEEN, LIKE, IN)를 사용할 수 있습니다.

SELECT name, salary
FROM employees
WHERE department = 'IT'
  AND salary > 3000000;

3. ORDER BY 정렬

ORDER BY 절은 결과를 특정 컬럼 기준으로 정렬합니다. ASC는 오름차순, DESC는 내림차순입니다. 기본값은 ASC(오름차순)입니다.

SELECT name, salary
FROM employees
ORDER BY salary DESC;

4. GROUP BY 집계

GROUP BY는 특정 컬럼을 기준으로 데이터를 그룹화합니다. COUNT, SUM, AVG, MAX, MIN 등의 집계 함수와 함께 사용하여 각 그룹에 대한 집계 결과를 얻습니다.

SELECT department, COUNT(*) AS cnt, AVG(salary) AS avg_sal
FROM employees
GROUP BY department;
